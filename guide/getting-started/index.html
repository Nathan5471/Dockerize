Dockerize Guide

Getting Started

Welcome to the Dockerize Guide! Here, you're going to find all of the information that you need to get started.

Join the Slack

The first step is to join #dockerize on the Hack Club Slack, if you have already. Haven't joined the Slack yet? No problem, you can join at hackclub.com/slack. Here you can stay up to date and find out about more super cool events run by Hack Club!

Setup Hackatime

Before you start making your project, you need to set up Hackatime. Hackatime is what Hack Club uses to track coding time. To set it up, go to hackatime.hackclub.com and following the guide for your favorite editor.

Create a Repo

The last step before you start coding is to create a repo for your code on a platform like GitHub. What's the point of building cool projects if you don't share them with the world? This is also required for you to be able to submit.
