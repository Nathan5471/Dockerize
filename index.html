Dockerize

You Ship an app with Docker
We Ship you a t-shirt!

1. Join #dockerize on Slack and checkout the guide (here)

2. Make your app. It should have a frontend, backend, and some way to store data.

3. Deploy your app with Docker using the guide found here.

4. Submit your app with a submission form that is coming soon!

5. Get your super awesome t-shirt designed by @candy on Slack!

Requirements:

  • You must be a highschooler
  • Your project should be open source and tracked with Hackatime (hackatime.hackclub.com)
  • Your project should be less than 30% AI and the README should be entirely written by you.
  • Your project should include a frontend, backend, and some way to store data. Got a different idea? Post it in #dockerize to get approval.
  • Your project should be deployed with Docker and include a Dockerfile and docker-compose.yaml file in your repo.
  • Your README should include instructions on how to run your app using Docker.
